Android TextView

Learn how to display text in your Android app

A TextView is a user interface element that displays text to the user. It is one of the most commonly used components in Android development.


TextView can display plain text or formatted text, and supports features like text styling, links, and more.

Example
activity_main.xml 
<TextView
    android:id="@+id/myText"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello World!"
    android:textSize="24sp" />
Output
Hello World!
